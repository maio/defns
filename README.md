# defns

A Clojure library/service...

inspired by http://erlang.org/pipermail/erlang-questions/2011-May/058768.html

## Usage

    (ns foo
      (:use defns))

    (defns deaccent)

    (println (deaccent "Příliš žluťoučký kůň úpěl ďábelské kódy."))
